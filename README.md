# CovidWard
Build a platform where the users can find information related to COVID-19 in the USA. They are be able to see the current overall statistics with graphs, see historic statistics with graphs showing data by dates (From start to now, or the user can add dates to filter), and filter by state.

La pagina se divide en dos:
●Today es en la cual se muestra la grafica con los datos del dia actual dependinedo del estado seleccionado
●Date es en la cual se muestra la grafica con los datos de un dia especifico a otro.

Cada parte cuenta con sus propios elementos los cuales ayudaran al usuario hacer su experiencia más placentera.



Requisitos para utilizar la app
●Tener NodeJS
●Se recomienda usar Chrome

Pasos para iniciar la app
●Descarga el archivo desde git
●En la linea de comandos, dirigete hacia la carpeta del proyecto
●cd frontend 
●npm start


Metas logradas:
●Implementación de Router de react
●Utilización de elementos de Material-UI v4
●Interfaz con elementos para el uso sencillo de la app
●Estilo minimalista
●Obtencion y manejo de la informacion de la API 
●Elementos usando la información de la API
●Clean code
